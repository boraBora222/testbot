## 2. Динамический задний фон

### Элементы фона:
1. **Grid Pattern** — анимированная сетка с пульсацией opacity
2. **Floating Orbs** — три размытых градиентных круга, плавающих с разной скоростью
3. **Geometric Shapes** — вращающиеся квадраты с эффектом дрейфа

### CSS-переменные для фона:
```css
--pattern-grid: rgba(99, 102, 241, 0.08);
--pattern-shape: rgba(99, 102, 241, 0.15);
```

### Анимации:
- `gridPulse` — 8s ease-in-out infinite
- `orbFloat` — 20s ease-in-out infinite (с разными задержками)
- `geoRotate` — 100-150s linear infinite
- `geoDrift` — 50-75s ease-in-out infinite

## Рекомендации по внедрению

1. **CSS-переменные**: Вынести все цвета в переменные для поддержки тем
2. **Tailwind**: Использовать @apply для повторяющихся стилей
3. **Framer Motion**: Для React-анимаций (fadeInUp, stagger)
4. **SVG-спрайт**: Объединить иконки в один файл
5. **Оптимизация**: Использовать will-change для анимаций
