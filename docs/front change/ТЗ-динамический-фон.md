# Техническое задание: Динамический фон

## 1. Общее описание

Динамический анимированный фон для веб-приложения CryptoDeal. Фон должен быть современным, ненавязчивым, визуально "живым", но не отвлекающим от контента.

---

## 2. Структура фона

Фон состоит из трёх независимых слоев, расположенных в следующем порядке (снизу вверх):

| Слой | Элемент | Z-index |
|------|---------|---------|
| 1 | Grid Pattern (сетка) | 0 |
| 2 | Floating Orbs (градиентные круги) | 1 |
| 3 | Geometric Shapes (геометрические фигуры) | 2 |

---

## 3. Слой 1: Grid Pattern (Пульсирующая сетка)

### 3.1 Визуальные параметры

| Параметр | Значение |
|----------|----------|
| Тип | Линейная сетка |
| Цвет линий | `rgba(99, 102, 241, 0.08)` |
| Размер ячейки | 60px × 60px |
| Толщина линий | 1px |

### 3.2 CSS-переменные

```css
--pattern-grid: rgba(99, 102, 241, 0.08);
```

### 3.3 Реализация

```css
.grid-pattern {
  position: absolute;
  inset: 0;
  background-image: 
    linear-gradient(to right, var(--pattern-grid) 1px, transparent 1px),
    linear-gradient(to bottom, var(--pattern-grid) 1px, transparent 1px);
  background-size: 60px 60px;
  animation: gridPulse 8s ease-in-out infinite;
}
```

### 3.4 Анимация

```css
@keyframes gridPulse {
  0%, 100% { opacity: 0.3; }
  50% { opacity: 0.6; }
}
```

| Параметр анимации | Значение |
|-------------------|----------|
| Длительность | 8s |
| Тип | ease-in-out |
| Повтор | infinite |
| Диапазон opacity | 0.3 → 0.6 → 0.3 |

## 7. Производительность

### 7.1 Требования

| Параметр | Требование |
|----------|------------|
| GPU-ускорение | Использовать transform и opacity |
| will-change | Добавить для анимированных элементов |
| Отключение на слабых устройствах | Через media query prefers-reduced-motion |

### 7.2 Оптимизация

```css
.orb,
.geo-shape {
  will-change: transform;
}

/* Отключение анимаций для пользователей с настройками доступности */
@media (prefers-reduced-motion: reduce) {
  .grid-pattern,
  .orb,
  .geo-shape {
    animation: none;
  }
}
```

## 8. Адаптивность

### 8.1 Планшеты (≤1024px)

- Сохранить все элементы
- Уменьшить размеры orb'ов на 20%
- Уменьшить размеры фигур на 15%

### 8.2 Мобильные (≤640px)

- Отключить Geometric Shapes (display: none)
- Уменьшить размеры orb'ов на 40%
- Уменьшить opacity сетки до 0.2

```css
@media (max-width: 640px) {
  .geo-shape {
    display: none;
  }
  
  .orb-1 { width: 300px; height: 300px; }
  .orb-2 { width: 240px; height: 240px; }
  .orb-3 { width: 180px; height: 180px; }
  
  .grid-pattern {
    opacity: 0.2;
  }
}
```
